**print Something**
```bash
echo hello wrold
```

**print Something to a file**
```bash
echo <text> >> <file_name>
```

**create new folder**
```bash
mkdir <folder_name>
```

**create new file**
```bash
touch <file_name>
```

**print working directory**
```bash
pwd
```

**change directory**
```bash
cd <directory_name>
```

**list all files and folders in current directory**
```bash
ls
```

**list all files and folders in current directory including files starting with .**
```bash
ls -a
```

**list all files and folders in current directory with more details**
```bash
ls -l
```

**see contents of file**
```bash
more <file_name>
```

**clear terminal**
```bash
clear
```

**copy file**
```bash
cp <file> <destination>
```

**copy folder**
```bash
cp -r <folder> <destination>
```

**move or rename file**
```bash
mv <file_name> <new_file_name or destination>
```

**remove file**
```bash
rm <file_name> 
```

**remove empty folder**
```bash
rmdir <folder_name> 
```

**remove folder and its content**
```bash
rm -r <folder_name> 
```

**view file tree**
```bash
find 
```

**view file tree of specific folder**
```bash
find <folder_name>
```

**find file or folder**
```bash
find -name <file_name or folder_name>
```
**Exit terminal**
```bash
exit
```

**view exit status of last command**
```bash
echo $?
```

**view manual of command**
```bash
man <command>
```

**print environment variables of shell**
```bash
printenv
```

**view all variables in the shell**
```bash
declare -p
```
